# Firmware and Software

This directory contains firmware and software for the ESL DUT dongle.

## Directory Structure

- `drivers/` - Device drivers and low-level software
- `tools/` - Command-line tools and utilities

## Integration

### OpenOCD Support
- FTDI FT2232H/FT4232H supports OpenOCD via MPSSE
- Enables JTAG/SWD debugging
- Console I/O via UART interfaces

### Power Monitoring
- INA219/INA228 drivers
- Integration with open-source power monitoring tools
- Python/C++ libraries

## Status

Firmware and software will be added as the project progresses.
